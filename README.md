# Superstore Profitability & Sales Performance Analysis

Süperstore satış ve kârlılık performans analizi projesi.

---

## Project Objective
## Proje Amacı

This project analyzes retail sales and profitability data using SQL and Tableau.

Bu proje, perakende satış ve kârlılık verilerini SQL ve Tableau kullanarak analiz etmektedir.

The goal of this project is to identify:
- profitable product categories
- the impact of discounts on profitability
- monthly sales trends
- business performance insights

Bu projenin amacı:
- kârlı ürün kategorilerini belirlemek
- indirimlerin kârlılık üzerindeki etkisini incelemek
- aylık satış trendlerini analiz etmek
- iş performansına yönelik içgörüler çıkarmaktır

---

## Tools Used
## Kullanılan Araçlar

- SQL (DuckDB)

- GitHub

---

## Key Insights
## Temel Bulgular

- Technology category generated the highest profit contribution.
- Teknoloji kategorisi en yüksek kâr katkısını sağlamıştır.

- Furniture category showed relatively low profitability.
- Mobilya kategorisinde kârlılık görece düşük kalmıştır.

- Higher discount levels were associated with lower average profit.
- Yüksek indirim oranları daha düşük ortalama kâr ile ilişkilendirilmiştir.

- Sales trends showed seasonal fluctuations throughout the year.
- Satış trendleri yıl boyunca dönemsel dalgalanmalar göstermiştir.
